# Kora-AI-Talent Project Overview: 
Kora is an intelligent Talent Agent designed to bridge the gap between high-volume recruitment and technical precision. Built for modern software agencies, Kora leverages Agentic AI to transform how founders and hiring managers evaluate technical talent.

Key Features
Deep Contextual Screening: Moves beyond keyword matching to understand a candidate's actual project contributions using Claude 3.5's reasoning.

Dynamic Interview Intelligence: Automatically generates 5 personalized technical questions based on the candidate's specific stack and past experience.

Automated Scoring: Provides a 0-100 'Match Score' against custom Job Descriptions to prioritize the top 5% of talent instantly.
Minimalist Executive Dashboard: A clean, distraction-free interface for rapid decision-making.
